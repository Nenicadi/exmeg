# Random Quote Generator

## Автор
Снопова Екатерина Андреевна

## Описание
Random Quote Generator - это GUI-приложение на Python для генерации случайных цитат с возможностью добавления собственных цитат, фильтрации и сохранения истории.

## Функциональность
- 🎲 Генерация случайных цитат из коллекции
- ➕ Добавление собственных цитат с указанием автора и темы
- 🔍 Фильтрация цитат по автору и теме
- 📋 Отображение истории всех цитат
- ❌ Удаление цитат из истории
- 💾 Автоматическое сохранение в JSON файл
- ✅ Валидация ввода (проверка на пустые строки)

## Требования
- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)
